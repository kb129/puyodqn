# ぷよぷよDQN - 仕様書

本プロジェクトは、本家ぷよぷよをコピーした実装に機械学習（DQN）による自動プレイヤーを組み込むことを目的としています。

## 🎯 プロジェクト概要

### 将来予定している機能
- **機械学習による最適手学習**: DQNを用いたぷよ配置の最適化
- **AI同士の同時学習**: 複数AIの競争による効率的学習
- **可変ゲームスピード**: 学習効率向上のための高速化対応

### 技術スタック
- **フロントエンド**: React + TypeScript + Canvas 2D
- **バックエンド**: Python + FastAPI (AI思考処理)
- **機械学習**: PyTorch/TensorFlow (将来実装)

## 🚀 開発フェーズ

### Phase 1: 基盤構築
1. フロントエンド基本実装 (React + Canvas)
2. Python通信サーバー (FastAPI + WebSocket)
3. データ交換テスト

### Phase 2: 無能CPU
1. 基本的な思考ロジック実装
2. フロントエンド・バックエンド連携

### Phase 3: ゲーム完成
1. 連鎖・スコア計算
2. おじゃまぷよシステム
3. 対戦機能

### Phase 4: 機械学習
1. DQNプレイヤー実装
2. 学習環境構築
3. 自動対戦・学習システム

## 📝 キー仕様

### ゲームの特徴
- **盤面**: 6列×12行（+隠し1行）
- **ぷよ**: 4色（赤・青・緑・黄）+ おじゃまぷよ
- **連鎖**: 4個以上の同色連結で消去
- **対戦**: ツモ共有、おじゃまぷよ攻撃

### プレイヤータイプ
- **人間プレイヤー**: キーボード操作
- **無能CPU**: 単純な深い谷埋めロジック
- **DQNプレイヤー**: 機械学習による最適化（将来実装）
